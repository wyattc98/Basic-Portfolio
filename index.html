After being introduced to JavaScript My desire to learn it spread like wildfire. I began with the Introduction to JavaScript course on Codecademy.com followed by the Introduction to JavaScript course on Both courses were able to build a strong foundation for me in JavaScript. Once my foundation was built I knew the only thing to do was to keep adding more stories to my foundation I applied to the UC Berkeley Coding Boot Camp and was accepeted! Thanks to the bootcamp I now have projects using things such as HandleBars, mySQL, Node JS, React, MongoDB, Express, jQuery, Axios, Cheerio, AJAX, JSON, APIS, and much more!
